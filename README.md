# 🚀 First .NET App 

Este repositório serve como um projeto de estudo e demonstração da minha transição técnica da linguagem **C (Base Académica)** para o ecossistema **.NET e C#**. 

O objetivo principal foi aplicar lógica de programação num ambiente moderno de desenvolvimento, utilizando ferramentas de **IA** para acelerar a curva de aprendizagem e implementar funcionalidades práticas.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** C#
* **Framework:** .NET (Console/Web SDK)
* **Ferramentas de Apoio:** IA (GitHub Copilot / ChatGPT) para pesquisa e refatoração.
* **Conceitos:** Programação Orientada a Objetos (POO), Manipulação de Dados e Estruturas de Repetição.

---

## 💡 Sobre o Projeto

Este projeto nasceu da necessidade de aplicar na prática os conceitos de sistemas de informação que desenvolvi na universidade. Como já possuía uma base sólida em **C**, utilizei este projeto para:
1.  Compreender a sintaxe e a estrutura de pastas do **.NET**.
2.  Implementar funcionalidades de gestão de dados (CRUD básico).
3.  Praticar a integração entre lógica de back-end e interação com o utilizador.

> **Nota de Aprendizagem:** "Aprender .NET após dominar a base de C permitiu-me focar menos na lógica base (que já dominava) e mais na arquitetura e nas bibliotecas poderosas que o C# oferece."

---

## 🚀 Como Executar

Se desejares testar o projeto localmente:

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/henriqueolivgp/.NET-App-Teste.git](https://github.com/henriqueolivgp/.NET-App-Teste.git)
    ```

2.  **Certifique-se de ter o SDK do .NET instalado.**

3.  **Execute a aplicação:**
    ```bash
    dotnet run
    ```
